## Notes

### Free Tier
- **No License Required**: Can use plugin with custom Google Cloud credentials (WordPress.org compliant)
- **Optional Free License**: Can activate free license for easier cloud server setup (no Google Cloud config needed)
- **Limited Features**: Single location only, basic styling
- **Setup Options**: 
  - Option 1: Use own Google Cloud credentials (no registration)
  - Option 2: Activate free license for cloud server access (easier setup)

### Pro Tier
- **License Required**: Must activate Pro license
- **Cloud Server**: MUST use cloud server (cannot bypass)
- **Full Features**: Multiple locations, advanced styling, analytics, etc.
- **No Custom Credentials**: Cannot use own Google Cloud Project

### Enterprise Tier
- **License Required**: Must activate Enterprise license
- **Flexibility**: Can use cloud server OR custom credentials
- **Full Features**: All Pro features + AI-powered review responses
- **Maximum Control**: Can use own Google Cloud Project if desired
